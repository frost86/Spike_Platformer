# Starlight Jump

Starlight Jump is an endless, procedurally generated 2D platformer built with HTML5 Canvas, Vanilla CSS, and JavaScript. Jump across platforms, collect stars to progress to the next level, and avoid the dangerous rotating spike balls!

## Features

- **Procedural Level Generation:** Every level is uniquely generated, increasing in difficulty as you progress.
- **Dynamic Entities:** Watch out for rotating spikes that patrol the platforms.
- **Progressive Difficulty:** The number of platforms, stars to collect, and enemies increase with each level.
- **Health System:** You have 3 lives, each allowing 3 hits. An intuitive UI displays your current health and lives.
- **Smooth Physics & Controls:** Enjoy responsive movement with gravity and jumping mechanics.

## How to Play

### Controls
**Desktop:**
- **Move Left:** `Arrow Left` or `A`
- **Move Right:** `Arrow Right` or `D`
- **Jump:** `Arrow Up`, `W`, or `Spacebar`

**Mobile / Touch:**
- **Move:** Touch anywhere on the screen and drag left or right. The game features a dynamic floating joystick that responds to the direction of your thumb.
- **Jump:** Quickly swipe upwards anywhere on the screen.

### Objective
Collect all the golden stars in the level to advance to the next stage. Avoid the red spike balls, or you will take damage. Lose all 3 lives, and it's game over!

## Installation & Running Locally

This game runs entirely in the browser and requires no backend server.

1. Clone the repository:
   ```bash
   git clone https://github.com/frost86/Spike-Platformer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Spike-Platformer
   ```
3. Open `index.html` in your favorite web browser.

## Tech Stack
- HTML5 Canvas for rendering graphics
- Vanilla CSS (`style.css`) for UI styling
- Vanilla JavaScript (`script.js`) for game logic
